## Login & Register UI

This is just a UI project for Login & Register pages. doesn't include any backend logic.

The forms are built using **react-hook-form** with **Yup validation schema**  
to keep the code clean, readable, and to reduce unnecessary logic.

---

##  Tech Stack
- React
- Vite
- react-hook-form
- Yup
- @hookform/resolvers

---

## How to Run

```bash
npm install
npm install react-hook-form yup @hookform/resolvers
npm run dev
